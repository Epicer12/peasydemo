# Peasy Website

Install Node.js (The reccommended one in the website)
v3 is the current version

```bash
git clone https://github.com/Epicer12/Peasy_Website.git
cd Peasy_Website
cd src/v1
cd src/v2
cd src/v3
```

```bash
npm install
```
Run just one time to install the dependancies

```bash
npm run dev
```
Run this is in either v1, v2 or v3
